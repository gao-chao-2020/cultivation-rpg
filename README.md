# 修仙 RPG · Cultivation RPG

键盘鼠标 **修仙主题** 挂机修炼游戏。打字 = 灵力，鼠标点击 = 炼丹，鼠标移动 = 身法，活跃时间 = 神识。在线排行榜。

## 功能

- 全局键盘/鼠标钩子，静默采集
- 8 个修仙境界（练气→飞升），彩色进度条
- 四维属性（灵力/身法/神识/炼丹）+ 修为
- 今日修炼统计
- 在线排行榜（Supabase，10 秒同步）
- 重生功能，清空数据从头再来
- 系统托盘，关闭窗口后台运行

## 快速开始

```bash
cd CultivationRPG
dotnet run
```

## 技术栈

C# .NET 9 + WPF · SQLite · Supabase · LiveCharts2 · Win32 Hooks

## 属性

| 属性 | 来源 | 系数 |
|------|------|------|
| 灵力 | 按键 | 1:1 |
| 身法 | 鼠标移动 | 像素/1000 |
| 炼丹 | 点击 | 1:1 |
| 神识 | 活跃(分) | 1:1 |

修为 = 四项总和
